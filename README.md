# qubit-simulator
A simple program that simulates a measurement taken on a **quantum register**.

## Description

### Fundamentals
```math
\lvert\psi\rangle = \alpha\lvert0\rangle + \beta\lvert1\rangle,
\lvert\alpha\rvert^{2} + \lvert\beta\rvert^{2} = 1
```
## How to use

### Dependencies
- python (language)
- python-numpy (package)
### Installing
- clone repository
### Running
```sh
$ python main.py 
```
